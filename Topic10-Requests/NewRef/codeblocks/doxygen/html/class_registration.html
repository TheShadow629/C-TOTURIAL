Detailed Description

It stores information about the student's units' result in the specified semester.

A student that has been registered will contain a student id, the semester, units.

The registration contains the list of units Results including its name, unit code, and marks. In addition to the student's ID, semester. It also displays the number of course taken by the student, and the student's total credit.
