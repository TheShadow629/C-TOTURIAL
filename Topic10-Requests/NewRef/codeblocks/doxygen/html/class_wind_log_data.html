Detailed Description

Manages all wind logs data in a vector. Wind logs data can be read from file and inputted into the vector. Available methods such as printing the wind logs for specific year and month.

Contains data of inputted wind logs and methods to print and write data to CSV file.

enum WindLogData::Mode

Represents the different operating modes available for data processing.

The available modes are SPEED for wind speed, TEMP for temperature, and SOLARRAD for solar radiation. These modes determine which data is processed and output by the WindLogData class.
